What is Circlegame?

Circlegame is a free to play 2D aim trainer. It is simplistic yet also so unique compared to other aim trainers out there. The goal in most of the gamemodes is to click as many circles as you can! The game comprises of 14+ gamemodes of varying levels of difficulty, meaning anyone can have fun while playing Circlegame!

Installing Circlegame!

Installing Circlegame is fairly easy and only takes a few minutes to complete. First, you must head to the Circlegame download page on Mediafire. Next, in the black box labeled "circlegame_v1.2.0", click on the blue download button labeled "DOWNLOAD (169.8MB)". Once the download is complete, you should see a circlegame_v1.2.0.zip file in your system's downloads folder. The final step is to right click on this file and click "Extract All". Once complete, the extracted folder should appear with the Circlegame application inside of it. For ease of use, we recommend that you create a desktop shortcut to the application, or even pin it to your taskbar!

Note: Circlegame is only available on Windows OS.

Help develop Circlegame!

Unfortunately, as of the release of v1.2.0 of Circlegame, the original creators of Circlegame have no intention of officially releasing new updates to the game in the near future. Therefore, we have decided that the best course of action to make Circlegame open-source and easily accessible for anyone to contribute if they so please. The Circlegame source code can be found in our GitHub repository, which also includes instructions on how code in Circlegame's development environment (Processing.org). Although you won't see your contributions to the game immediately be implemented, be sure that one day we will review your pull requests and implement your changes into the next release!
